## Usage guide

#### Install: 

`yarn`

#### Deployment:
1. `yarn deploy --tags UniswapV2Factory`
1. `yarn deploy --tags UniswapV2Router02`
2. `yarn deploy --tags Multicall2`
3. `yarn deploy --tags DummyTokens`

#### Debug:

Modify the `scripts/debug.js` scripts and run `yarn debug`.
